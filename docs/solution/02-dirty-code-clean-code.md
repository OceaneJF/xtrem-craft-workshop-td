# Clean code VS Dirty code

## Qu'est-ce que du code sale ?

- Un marécage ou on se perd
- Un code spaghetti
- Du code sans structure 
- Du code dupliqué
- Un patchwork mille fois reprisé

## Qu'est-ce que du code propre ?

- Lisible => se lit comme une phrase
- Elégant => Assez simple et direct pour ne pas laisser de place au bug
- Facile à modifier => Peut être modifié par un autre que son auteur
- Écrit avec soin => aucun problème évident ne saute aux yeux 
- Sans surprise => Le corps des méthodes est ce qu'on avait deviné
